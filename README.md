==================================================
   MANUAL DE CONFIGURACIÓN - BARBERÍA ELITE
==================================================

Este proyecto es una plantilla web profesional de 4 páginas 
(Inicio, Servicios, Galería y Contacto) optimizada para móviles.

--------------------------------------------------
1. CÓMO CAMBIAR EL TELÉFONO / WHATSAPP
--------------------------------------------------
Debes buscar en los 4 archivos HTML la línea que contiene:
href="https://wa.me/56926073767"

Cambia el número (después del 569) por el del cliente. 
No borres el "https://wa.me/".

--------------------------------------------------
2. CÓMO CAMBIAR LOS PRECIOS
--------------------------------------------------
Abre el archivo "servicios.html". 
Busca las etiquetas que tienen la clase: class="price-tag".
Ejemplo: <span class="price-tag">$8.000</span>
Solo cambia el número dentro del texto.

--------------------------------------------------
3. CÓMO CAMBIAR LAS FOTOS DE LA GALERÍA
--------------------------------------------------
Abre "galeria.html".
Busca las etiquetas <img>. 
Cambia el texto dentro de src="..." por el nombre de tu foto.
Ejemplo: <img src= Imagenes/images1.jpg"> 
(Asegúrate de que la foto esté en la misma carpeta).

--------------------------------------------------
4. CONFIGURACIÓN DEL MENÚ MÓVIL (IMPORTANTE)
--------------------------------------------------
Si agregas una página nueva, debes añadir el link en el 
<nav class="nav-container"> de TODOS los archivos HTML para
que la navegación no se rompa.

--------------------------------------------------
5. CONTACTO 
--------------------------------------------------
En "contacto.html"
